Домашнее задание №22: Теги и Фильтры шаблонов

1. Встроенные фильтры для строк

Оригинальные данные из Python Примененный фильтр Django Результат в HTML
{{ student_name }} |title (Первые буквы заглавные) {{ student_name|title }}
{{ project_description }} |upper (ВСЕ БУКВЫ ЗАГЛАВНЫЕ) {{ project_description|upper }}
{{ student_name }} |length (Подсчет символов) Длина строки: {{ student_name|length }} знаков

2. Дополнительные фильтры (Дата и Срезы)

Оригинальная дата из базы: {{ current_date }}

Отформатированная дата (фильтр |date): {{ current_date|date:"d E Y года в H:i" }}

Оригинальный текст: {{ project_description }}

Фильтр |slice:":17" (Отрезать первые 17 символов): {{ project_description|slice:":17" }}...

3. Встроенные теги шаблонов (Логика)

Проверка тега if для оценки ({{ score }} баллов):

{% if score > 50 %}

Отличный результат! Задание выполнено.

{% else %}

Требуется доработка.

{% endif %}

Вывод списка через тег for:

    {% for skill in skills_list %}
  • Навык №{{ forloop.counter }}: {{ skill }}
    • {% endfor %}

Проверка тега empty на пустом списке:

    {% for item in empty_list %}
  • {{ item }}
    • {% empty %}
  • (Тег сработал: Список элементов полностью пуст)
    • {% endfor %}
